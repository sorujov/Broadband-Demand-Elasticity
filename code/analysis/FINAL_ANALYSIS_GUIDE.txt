STEP 8: ROBUSTNESS CHECKS (20-30 minutes)
================================================================================

PURPOSE:
--------
Validate that your main findings hold across:
- Alternative specifications
- Different samples
- Various treatments

WHY IMPORTANT:
-------------
- Referees will ask: "Are results sensitive to specification?"
- Shows findings are not driven by arbitrary choices
- Increases confidence in main results
- Standard practice in top journals

WHAT IT DOES:
------------
✓ Check 1: Alternative dependent variables
  - Bandwidth usage (baseline)
  - Broadband subscriptions  
  - Subscriptions per 100 (levels)

✓ Check 2: Different time periods
  - Full sample (2010-2023)
  - Pre-COVID (2010-2019)
  - Post-crisis (2012-2023)
  - Recent years (2015-2023)

✓ Check 3: Subsample analysis
  - EU countries only
  - EaP countries only
  - Compare elasticities

✓ Check 4: Outlier treatment
  - No treatment (baseline)
  - Winsorized at 1-99%
  - Trimmed extreme 1%

✓ Check 5: Alternative controls
  - Minimal (price only)
  - Baseline (price + GDP)
  - Extended (+ internet users)
  - Full (+ urbanization + density)

HOW TO RUN:
-----------
cd code\analysis
python 08_robustness_checks.py

OUTPUTS CREATED:
---------------
results/robustness/:
- robustness_alternative_depvars.txt
- robustness_time_periods.txt
- robustness_subsamples.txt
- robustness_outliers.txt
- robustness_controls.txt

results/tables/:
- robustness_summary.txt

EXPECTED TIME: 20-30 minutes (runs 5 sets of regressions)

WHAT TO LOOK FOR:
----------------
1. CONSISTENT RESULTS = ROBUST
   - Price elasticity stays in similar range
   - Same sign across specifications
   - Significance pattern consistent

2. INCONSISTENT RESULTS = SENSITIVE
   - Elasticity varies widely
   - Significance changes across specs
   - May need to explain or use preferred spec

TYPICAL ROBUSTNESS PATTERN:
---------------------------
Main result: -0.009 (baseline, FE Two-Way)

Robust if you see:
- Alt dep vars: -0.007 to -0.012
- Time periods: -0.008 to -0.011
- Subsamples: EU -0.008, EaP -0.012
- Outliers: -0.009 to -0.010
- Controls: -0.008 to -0.011

→ CONCLUSION: "Results are robust across specifications"

Not robust if you see:
- Wide range: -0.001 to -0.050
- Sign changes: Some positive, some negative
- Significance varies: Some significant, others not

→ CONCLUSION: "Results are sensitive to specification choice"

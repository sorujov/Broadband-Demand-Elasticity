# Broadband Price Elasticity Analysis

## Project Structure

- **data/**: All data files
  - raw/: Original downloaded data
  - interim/: Intermediate processing steps
  - processed/: Final analysis-ready datasets
- **code/**: All analysis scripts
  - data_collection/: Download scripts
  - data_preparation/: Cleaning and preparation
  - analysis/: Econometric models
  - visualization/: Plots and figures
- **results/**: Analysis outputs
- **docs/**: Documentation and manuscript
- **figures/**: All generated figures
- **logs/**: Execution logs
